## AWS Services Overview

### Amazon Elastic MapReduce (EMR)

Amazon EMR allows the running of big data frameworks such as Apache Hadoop and Apache Spark on AWS. It simplifies the processing of vast amounts of data across resizable clusters of Amazon EC2 instances.

### Amazon QuickSight

QuickSight is a scalable, serverless, embeddable machine learning-powered business intelligence service for publishing interactive dashboards.

### Amazon AppFlow

AppFlow is a fully managed integration service that enables secure data transfer between SaaS applications like Salesforce, Slack, ServiceNow, etc., and AWS services.

### Amazon EventBridge

EventBridge is a serverless event bus that simplifies application integration using data from your applications, AWS services, and SaaS applications.

### AWS App Runner

App Runner allows for easy deployment and running of web applications, API services, backends, and websites on AWS without worrying about infrastructure or container orchestration. It supports direct deployment from container images or source code.

### AWS App2Container

App2Container is a command-line tool for modernizing .NET and Java applications into containerized applications, facilitating easier deployment and scaling.

### AWS Copilot

Copilot is a command-line tool that helps quickly launch and easily manage containerized applications on AWS, streamlining the deployment process.

### AWS Elastic Kubernetes Service (EKS) Deployment Options

- **AWS EKS Cluster**: Fully managed Kubernetes service.
- **AWS Outposts**: Runs Kubernetes workloads on physical racks in your on-premises network.
- **AWS Anywhere**: Extend AWS infrastructure, APIs, and tools to any data center, co-location space, or on-premises facility for a truly consistent hybrid experience.
- **AWS Distro**: Offers a distribution of the same open-source Kubernetes software deployed by Amazon EKS in the AWS cloud.

### Red Hat OpenShift Service on AWS (ROSA)

ROSA provides a fully managed Red Hat OpenShift platform on AWS with a pay-as-you-go billing option, operated by Red Hat and AWS.

### AWS Database Migration Service (DMS)

DMS facilitates the quick and secure migration of databases to AWS, supporting both homogeneous and heterogeneous migrations.

### Amazon DocumentDB

DocumentDB is a fast, scalable, highly available MongoDB-compatible database service.

### Amazon MemoryDB for Redis

MemoryDB for Redis is a Redis-compatible, durable, in-memory database service in the AWS Cloud.

### Amazon ElastiCache

ElastiCache offers fully managed in-memory data store services, based on Redis or Memcached, to power real-time applications.

### AWS Cloud Development Kit (CDK)

The AWS CDK is an open-source software development framework for defining cloud infrastructure in code and provisioning it through AWS CloudFormation.

### AWS CloudShell

CloudShell is a browser-based shell available within the AWS Management Console, providing direct access to AWS services.

### AWS Artifact

Artifact is a fully managed service for storing, publishing, and distributing software packages and artifacts.

### AWS CodeStar

CodeStar provides integrated tools for developing, building, and deploying applications on AWS quickly and easily.

### AWS CodeGuru

CodeGuru offers intelligent recommendations for improving application performance, efficiency, and code quality through automated code reviews and profiling.

### AWS Fault Injection Simulator (FIS)

FIS is a managed service that enables the performance of fault injection experiments on AWS workloads to improve resilience and reliability.

### AWS Control Tower

Control Tower assists in setting up and governing a secure, multi-account AWS environment, automating account provisioning and ongoing policy management.

### Amazon Lookout for Metrics

Part of the Amazon Lookout family, this service detects anomalies in business metrics using machine learning, improving operational insight.

### AWS Compute Optimizer

Compute Optimizer aids in optimizing computing capacity and costs by recommending the right sizing for EC2 instances based on usage patterns.

### Amazon FSx

Amazon FSx provides fully managed third-party file systems with several options:

- **Amazon FSx for Lustre**: High-performance file system optimized for fast processing of workloads.
- **Amazon FSx for Windows File Server**: Fully managed Windows native file system.
- **Amazon FSx for NetApp ONTAP**: Managed NetApp ONTAP file system with native data management and protection.
- **Amazon FSx for OpenZFS**: Managed file system based on the OpenZFS standard.

### AWS Backup

AWS Backup is a fully managed backup service that automates backup across AWS services, offering policies for retention and lifecycle management.

### AWS Proton

Proton automates container and serverless application deployment, simplifying infrastructure management for microservices architectures.

### AWS CloudHSM

CloudHSM offers hardware security modules in the cloud, allowing customers to generate and use their encryption keys securely.

### AWS Network Firewall

A fully managed network firewall service for Amazon VPC, providing protection against malicious web traffic and enforcing network traffic rules.

### Amazon Detective

Detective simplifies the analysis and investigation of security issues, helping quickly identify the root cause of potential security threats.

### AWS VPN

VPN enables secure connections between on-premises networks and AWS, extending your network into the cloud with encrypted connections.

### AWS PrivateLink

PrivateLink allows private connectivity to AWS services, keeping traffic within the AWS network and away from the public internet.

### AWS License Manager

License Manager simplifies the management of software licenses from various vendors, helping ensure compliance with license terms.

### AWS Health Dashboard and Health API

The Health Dashboard and API provide real-time visibility into the status and health of AWS services, enhancing operational awareness and incident response.

### AWS Elastic Beanstalk

Elastic Beanstalk is an easy-to-use service for deploying applications which automatically handles the deployment, from capacity provisioning, load balancing, and auto-scaling to application health monitoring.
